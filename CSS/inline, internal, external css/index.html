Type of css

Inline CSS
CSS সরাসরি HTML tag-এর ভিতরে style attribute ব্যবহার করে লেখা হলে তাকে Inline CSS বলে।

Example:

This is inline CSS


Internal CSS
HTML file-এর head এর ভিতরে style tag ব্যবহার করে CSS লেখা হলে তাকে Internal CSS বলে।

Example:

This is internal CSS


External CSS (Most Professional)
CSS আলাদা .css file-এ লিখে HTML এর সাথে link করলে তাকে External CSS বলে।
