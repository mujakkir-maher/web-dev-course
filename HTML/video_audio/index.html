topic 01: video

attributes of videos are given velow

src → ভিডিও ফাইলের path নির্ধারণ করে
controls → play, pause, volume ইত্যাদি control button দেখায়
autoplay → page load হলে ভিডিও নিজে থেকে চালু হয়
muted → ভিডিওর sound বন্ধ থাকে
loop → ভিডিও শেষ হলে আবার শুরু হয়
poster → ভিডিও শুরু হওয়ার আগে thumbnail image দেখায়
width → ভিডিওর প্রস্থ নির্ধারণ করে
height → ভিডিওর উচ্চতা নির্ধারণ করে
preload → ভিডিও কতটুকু আগে থেকে load হবে (auto, metadata, none)
playsinline → mobile-এ fullscreen না হয়ে inline-এ play করে
crossorigin → অন্য domain থেকে ভিডিও লোড করলে CORS নিয়ন্ত্রণ করে
controlslist → নির্দিষ্ট control (যেমন download) hide করতে ব্যবহৃত হয়

topic 02: audio

no audio found

attributes of audio are given below

src → Audio file-এর path
controls → Play/Pause/Volume button দেখায়
autoplay → Page load হলে নিজে থেকে play শুরু করে
muted → Sound বন্ধ থাকবে
loop → Audio শেষ হলে আবার শুরু হবে
preload → আগে থেকে কতটুকু load হবে (auto, metadata, none)
more ab0ut preload: HTML-এর preload attribute হলো audio এবং video tag-এর জন্য ব্যবহৃত একটি attribute যা browser-কে নির্দেশ দেয় page load-এর সময় ভিডিও বা অডিও কতটুকু আগে থেকে load করবে।

none : কোনো data load করা হবে না। শুধুমাত্র play করলে load হবে। fastest load, bandwidth save
metadata : শুধুমাত্র media-এর metadata load হবে (duration, dimensions, tracks)।
auto : browser চেষ্টা করবে পুরো media load করতে।

crossorigin → অন্য domain থেকে audio load করলে CORS নিয়ন্ত্রণ

topic 03: SVG

SVG: একটি vector graphics format যা 2D graphics (shape, line, curve, text, icons) তৈরি করতে ব্যবহৃত হয় এবং scale করলে quality হারায় না।

topic 04: iFrame

iframe (Inline Frame): এটি মূল page-এর ভিতরে একটি "window" তৈরি করে, যেখান থেকে অন্য URL load করা যায়।
